# Assets Klasörü

Bu klasöre logo dosyalarınızı ekleyin:

- `logo.png` veya `logo.jpg` - Ana logo (önerilen: 512x512 veya daha büyük)
- `icon.ico` - Windows icon dosyası (256x256 veya daha büyük)
- `icon.png` - Electron icon (512x512 veya daha büyük)

Logo dosyasını buraya ekledikten sonra, Electron main.js dosyasında icon path'ini aktif edin.
